L'histoire du Web

La différence entre le Web et internet

Internet est un réseau d'ordinateurs interconnectés. A l'origine, internet s'appelait "Arpanet", il reliait 4 universités de la côte ouest des états-unis.

Le Web, quant à lui, fait référence au World Wide Web (W.W.W.), qui est un système qui permet d'accéder aux ressources présentes sur internet et de naviguer entre différentes pages via des liens hypertextes. Il à été créé par le Britannique Tim Berners Lee le 12 Mars 1989 au CERN (Organisation européenne pour la recherche nucléaire).

Les 3 technologies fondatrices du Web sont :

  • HTML (HyperText Markup Language), un langage de balises qui sert au formatage des pages Web.
  • URL (Uniform Resource Locator), les adresses uniques pour identifier chaques ressources sur le Web.
  • HTTP (HyperText Transfer Protocole), qui sert à établir une liaison entre un client et un serveur.

Le Web 1.0

Le Web 1.0 c'est l'accès à l'information en temps réel, les emails, les forums et les chat.

Les pages Web étaient statiques, c'est-à-dire qu'elles n'étaient disponibles qu'en lecture seule. L'un créait, l'autre lisait. L'un créait, l'autre achetait...

C'était du contenue créé par des professionnels pour des particuliers.

Le Web 2.0

Le Web 2.0 c'est l'explosion du contenu créé par les utilisateurs (notamment grâce aux réseaux sociaux).

C'est également le début du e-commerce et de la gig economy (économie des petits boulots) via des applications comme Uber et Uber-eat.

La croissance du Web 2.0 est étroitement liée à l’invention du smartphone. En effet, le smartphone permet d’utiliser internet n’importe ou, sans câble, plus besoin d’être assis devant sa machine.

Avec le Web 2.0, les pages Web sont dynamiques.

Les problèmes du Web 2.0

Le problème avec le Web 2.0 c'est qu'à l'origine le Web devait être décentralisé, ouvert, libre d'accès et résistant à la censure. Mais c'est le contraire qui s'est produit :

  • Une énorme partie du contrôle se retrouve entre les mains des géants du Web (GAFAM). Les données utilisateurs sont collectées et monétisées à hauteur de milliards de dollars par an, sans que les utilisateurs n'en tirent aucun profit.
  • Les plateformes qui contrôlent les utilisateurs peuvent choisir de bannir et de censurer leurs utilisateurs ou des contenus qui ne leur conviennent pas.
  • Et la centralisation qui pose problème, qui rend les attaques plus faciles mais aussi le contrôle par des gouvernements autoritaires comme la Chine par exemple. C'est pourquoi le Web 3.0 commence à voir le jour.
